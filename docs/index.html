Welcome

This course teaches statistics the way research actually works starting with your data, not a textbook formula. Every method is introduced with a practical scenario, explained in plain language, and then implemented step-by-step in R.

The course is designed for PhD students in biomedical and clinical research. That includes people working with human patient data, animal experiments, survey data, or longitudinal studies. You do not need to be an expert in R or statistics to start, but you do need R and RStudio installed on your computer.

ImportantStart Here: Install R and Course Packages

Before opening any session, go to Before You Begin: Setup and follow the setup steps there. You will install R, RStudio, and all the packages the course uses. This takes about 10–15 minutes and only needs to be done once.

How the Course Is Organized

The course has four parts. Each part builds on the previous one.

TipChoose Your Starting Point
Your background Where to start
New to statistics Part 1 work through the course in order
Familiar with R, new to modeling Part 1 then continue to Part 2
Interested in regression and prediction Part 2 after reviewing Part 1 if needed
Looking for advanced methods only Part 3 after the core concepts in Parts 1 and 2
Need a quick method guide Part 4 for the reference and resources

If you are unsure where to begin, start with Part 1 and follow the course sequentially.

TipYour Quick Reference

The Statistical Decision Guide in Part 4 is a reference card you can return to at any point. It tells you which method to use for common research questions and links back to the relevant sessions.

Part 1: Inference — Testing and Estimation

This part introduces the foundations of statistical inference and hypothesis testing. You will learn how to interpret p-values, confidence intervals, and effect sizes before moving on to regression and modeling.

  • Sampling, confidence intervals, and the central limit theorem
  • Hypothesis testing and the logic of evidence
  • t-tests for comparing means
  • Non-parametric alternatives when assumptions fail
  • ANOVA for comparing multiple groups
  • Categorical tests such as chi-square and Fisher’s exact
  • Power and sample size planning

Part 2: Regression and Modeling

This part covers the most commonly used predictive models in biomedical research. You will learn how to fit models, interpret coefficients, check assumptions, and report results clearly.

  • Correlation and association
  • Linear regression for continuous outcomes
  • Multiple regression for adjusted effects
  • Logistic regression for binary outcomes
  • Model building, diagnostics, and validation
  • Missing data handling with multiple imputation
  • Mixed models for repeated measures and clustered data

Part 3: Advanced Methods

This part introduces advanced analytic methods for more complex study designs and causal questions.

  • Survival analysis and time-to-event data
  • Causal inference using DAGs and observational data
  • Mendelian randomization for genetic instrumental variables

Part 4: Reference and Resources

This part gives you tools to apply the methods thoughtfully and find answers when you need them.

  • Statistical Decision Guide for method selection
  • Clinical research methods and study design considerations
  • Help and documentation for using the course materials

Prerequisites

  • R and RStudio installed on your computer
  • Basic ability to run R code in RStudio
  • No prior statistics knowledge is assumed beyond high-school level

How to Use This Course

Each session follows the same structure:

  1. When do you use this? A concrete scenario from your field
  2. Background Why the method works, in plain language
  3. Worked examples Step-by-step R code on real data
  4. What can go wrong Common mistakes and how to avoid them
  5. Exercises Practice problems with solutions
  6. Comprehension check Self-check questions

You can work through the course week by week, or jump to a specific method when you need it. If you are unsure which method to use, go to the Statistical Decision Guide in Part 4.
