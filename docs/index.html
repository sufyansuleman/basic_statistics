Course Overview

Welcome to the Basic Statistics and Interpretation course. This course is designed to build your foundational understanding of statistical concepts, data summarization, inference, and practical interpretation of results.

The course focuses on clear reasoning, effective data communication, and the ability to interpret findings in context rather than just compute statistics. You will learn how to explore real data, assess variation, and draw useful conclusions that support evidence-based decisions.

Earlier introductory material on R programming fundamentals and the broader clinical research analytics track is archived separately, so learners can focus on the current course’s statistical and interpretation path. The course starts with basic statistics and builds a structured bridge to interpretation, communication, and reproducible analysis.

What You Will Learn

  • Understand central descriptive statistics and graphical summaries.
  • Interpret variability, probability, and basic sampling ideas.
  • Use confidence intervals and hypothesis tests appropriately.
  • Explain statistical findings in plain language.
  • Connect results to real questions with reasoned interpretation.
  • Build reproducible analysis workflows with Quarto and renv.
  • Create effective visualizations that support statistical conclusions.

Methods covered in this course

This course explicitly teaches the statistical methods required for basic applied data analysis, including:

  • descriptive statistics and data summaries
  • data transformation and standardization
  • confidence intervals and uncertainty
  • t-tests and group comparison tests
  • power and sample size reasoning
  • correlation and regression modeling
  • ANOVA and multiple comparisons
  • chi-square tests for categorical data
  • logistic regression for binary outcomes
  • survival analysis and time-to-event modeling
  • model diagnostics and interpretation

Why This Course

This course is designed to make basic statistics accessible and useful for interpretation. It helps learners go beyond calculations so they can understand what results mean, when methods apply, and how to communicate findings clearly.

R is the leading language for statistical analysis. With modern R packages for data manipulation and reproducible reporting, you can build workflows that support clear conclusions and trustworthy interpretation.

Aims

  1. Build confidence in reading and interpreting statistical results.
  2. Connect quantitative findings to clear research conclusions.
  3. Use visualization and summary statistics to communicate uncertainty.
  4. Develop reusable, reproducible analysis workflows in R.
  5. Explain what results mean and when they are useful.

Learning Objectives

Upon completing this course, you will be able to:

  1. Interpret descriptive statistics, variation, and relationships in data.
  2. Explain uncertainty using confidence intervals, error bands, and plots.
  3. Create figures that answer research questions and support conclusions.
  4. Organize analysis code and results in reproducible Quarto documents.
  5. Summarize statistical findings in plain language for a broad audience.
  6. Assess whether a result is meaningful and how to communicate it clearly.

Prerequisites

  • Comfort with basic statistics and research study concepts.
  • Familiarity with the RStudio interface is helpful but not required.
  • A computer with internet access and the ability to install R packages.

Please follow the installation and environment setup instructions in the course materials before beginning the exercises.
