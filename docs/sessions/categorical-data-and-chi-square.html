14  Categorical Data and Chi-Square Tests

14.1 When Do You Use This?

You want to know whether adverse event rates differ between two treatment arms in a clinical trial ? both outcomes are binary (event: yes/no). Or you want to know whether disease stage at diagnosis differs between men and women. These are questions about counts and proportions, not means, and they call for chi-square or Fisher’s exact tests.

14.2 Learning Objectives

After completing this session you will be able to:

  • Identify when chi-square or Fisher’s exact test is the right choice
  • Construct a contingency table and calculate expected cell counts
  • Run chisq.test() and fisher.test() in R and interpret the output
  • Use prop.test() to compare two proportions with a confidence interval
  • Build publication-quality frequency tables with gtsummary
  • Know when to apply Yates’ continuity correction

14.3 Background: Testing Categorical Associations

When both variables are categorical, we test whether the distribution of one variable differs across levels of the other ? or equivalently, whether the two variables are associated.

The 2?2 contingency table:

Outcome: Yes Outcome: No Row total
Group A \(a\) \(b\) \(a+b\)
Group B \(c\) \(d\) \(c+d\)
Column total \(a+c\) \(b+d\) \(n\)

Chi-square test: Under independence, each cell’s expected count is

\[E_{ij} = \frac{\text{row total}_i \times \text{col total}_j}{n}\]

The test statistic is \(\chi^2 = \sum \frac{(O-E)^2}{E}\), with degrees of freedom \((r-1)(c-1)\).

Fisher’s exact test: Used when expected cell counts are small (< 5). It computes the exact probability of seeing a table as extreme as observed under independence.

Rule of thumb: Use chi-square when all expected counts = 5; use Fisher’s exact when any expected count < 5.

14.4 Example 1: Clinical Data (PBC)

14.4.1 Is Sex Associated with Disease Stage?

pbc <- survival::pbc %>%
  as_tibble() %>%
  janitor::clean_names() %>%
  mutate(
    sex   = factor(sex, levels = c("f", "m"), labels = c("Female", "Male")),
    trt   = factor(trt, levels = c(1, 2), labels = c("D-penicillamine", "Placebo")),
    stage = factor(stage, levels = 1:4, labels = paste("Stage", 1:4)),
    died  = as.integer(status == 2)
  )
# Contingency table
table(pbc$sex, pbc$stage)
        
         Stage 1 Stage 2 Stage 3 Stage 4
  Female      18      84     139     127
  Male         3       8      16      17
chisq.test(table(pbc$sex, pbc$stage))
Warning in chisq.test(table(pbc$sex, pbc$stage)): Chi-squared approximation may
be incorrect

    Pearson's Chi-squared test

data:  table(pbc$sex, pbc$stage)

A significant chi-square test tells you that the distribution of stage differs between men and women (or equivalently, sex and stage are associated). It does not tell you the direction or size.

Visualise:

pbc %>%
  filter(!is.na(sex), !is.na(stage)) %>%
  ggplot(aes(x = stage, fill = sex)) +
  geom_bar(position = "fill") +
  scale_fill_manual(values = c("steelblue", "tomato")) +
  scale_y_continuous(labels = scales::percent) +
  labs(title = "Disease stage by sex",
       x = NULL, y = "Proportion", fill = "Sex") +
  theme_bw()

14.4.3 Comparing Two Proportions with a CI

prop.test() compares two proportions and gives a confidence interval for the difference.

# Proportion who died in each treatment group
pbc_rand <- pbc %>% filter(!is.na(trt), !is.na(died))

deaths_by_trt <- pbc_rand %>%
  group_by(trt) %>%
  summarise(n = n(), deaths = sum(died))

prop.test(x = deaths_by_trt$deaths, n = deaths_by_trt$n)

    2-sample test for equality of proportions with continuity correction

data:  deaths_by_trt$deaths out of deaths_by_trt$n
X-squared = 0.076731, df = 1, p-value = 0.7818
alternative hypothesis: two.sided
95 percent confidence interval:
 -0.09334697  0.13691100
sample estimates:
   prop 1    prop 2 
0.4113924 0.3896104

Interpretation: The 95% CI for the difference in proportions includes/excludes zero, indicating whether the groups differ in mortality.

14.5.2 Relative Risk and Odds Ratio

For a 2?2 table, you can calculate the relative risk (RR) and odds ratio (OR) by hand:

a <- tab_mel["Male", "Yes"]
b <- tab_mel["Male", "No"]
c <- tab_mel["Female", "Yes"]
d <- tab_mel["Female", "No"]

rr <- (a / (a + b)) / (c / (c + d))
or <- (a * d) / (b * c)

cat("Relative Risk:", round(rr, 2), "\n")
Relative Risk: 1.46 
cat("Odds Ratio:   ", round(or, 2), "\n")
Odds Ratio:    2.01

For logistic regression-based OR with CIs, see the Logistic Regression session.

14.7 What Can Go Wrong

Using chi-square with small expected counts When any expected cell count is < 5, the chi-square approximation is unreliable. Use Fisher’s exact test. The output of chisq.test() warns you when this occurs.

Confusing counts with proportions chisq.test() requires a table of counts, not proportions. Dividing cells by totals before running the test gives incorrect results.

Yates’ continuity correction R applies Yates’ correction by default for 2?2 tables (correct = TRUE). This makes the test more conservative. It is often dropped in practice; use correct = FALSE to match Pearson’s chi-square, but be consistent in your reporting.

Interpreting significance as strong association With large samples, chi-square detects tiny, clinically irrelevant differences. Supplement the p-value with a measure of effect size: Cram?r’s V (for tables), or relative risk / odds ratio (for 2?2 tables).

Forgetting that chi-square is non-directional A significant chi-square tells you the variables are associated ? not which group has a higher proportion. Always report the contingency table or proportions alongside the test.

14.9 Comprehension Check

  1. When should you use Fisher’s exact test instead of a chi-square test?
  2. You run chisq.test() and R prints a warning: “Chi-squared approximation may be incorrect.” What does this mean, and what should you do?
  3. A chi-square test gives p = 0.001. Does this tell you which group has a higher proportion of events?
  4. What is the difference between a relative risk (RR) and an odds ratio (OR)? When are they approximately equal?
  5. You have a 3?4 contingency table (3 groups, 4 ordered categories). What are the degrees of freedom for the chi-square test?
Answers
  1. When any expected cell count is < 5. Fisher’s exact test does not rely on the large-sample chi-square approximation and is valid for any sample size.
  2. The warning means at least one expected cell count is < 5, so the chi-square approximation is unreliable. Switch to fisher.test().
  3. No. Chi-square is a non-directional test ? it tells you whether an association exists, not its direction or size. You must examine the contingency table or plot the proportions to see which group has higher rates.
  4. Relative risk = (proportion in group A) / (proportion in group B). Odds ratio = (odds in group A) / (odds in group B). They are approximately equal when the event is rare (< 10?15% in both groups). For common outcomes, the OR exaggerates the association relative to the RR.
  5. Degrees of freedom = (rows - 1) ? (columns - 1) = (3 - 1) ? (4 - 1) = 6.
