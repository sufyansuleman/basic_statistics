12  Nonparametric Tests

12.1 When Do You Use This?

You have only 8 mice per group and the outcome variable (e.g., tumour volume) is heavily right-skewed with an obvious outlier. A t-test requires approximate normality, and with \(n = 8\) you cannot rely on the CLT. Nonparametric tests work on the ranks of the data rather than the raw values, so they are robust to non-normality and outliers.

12.2 Learning Objectives

After completing this session you will be able to:

  • Identify when a nonparametric test is more appropriate than a t-test
  • Perform the Wilcoxon rank-sum (Mann-Whitney) and Wilcoxon signed-rank tests in R
  • Perform the Kruskal-Wallis test as a nonparametric ANOVA alternative
  • Interpret the results of nonparametric tests
  • Understand the trade-offs: nonparametric tests are more robust but less powerful when normality holds

12.3 Background: Ranks Instead of Raw Values

Parametric tests (t-tests, ANOVA) make assumptions about the distribution of the data. Nonparametric tests replace raw values with their ranks and test whether the distribution of ranks differs between groups.

Key advantage: robust to outliers and skewed distributions. Key cost: slightly less powerful than parametric tests when the normality assumption is met.

Parametric test Nonparametric equivalent When to use
One-sample t-test Wilcoxon signed-rank test n small, distribution non-normal
Two-sample t-test Wilcoxon rank-sum (Mann-Whitney) Two independent groups, n small or skewed
Paired t-test Wilcoxon signed-rank test Paired data, n small or non-normal
One-way ANOVA Kruskal-Wallis test ≥ 3 groups, normality violated

12.4 Example 1: Clinical Data (PBC)

Bilirubin is heavily right-skewed and contains extreme outliers. For a subgroup analysis with small numbers, a nonparametric test is prudent.

pbc <- survival::pbc %>%
  as_tibble() %>%
  janitor::clean_names() %>%
  mutate(
    sex   = factor(sex, levels = c("f", "m"), labels = c("Female", "Male")),
    trt   = factor(trt, levels = c(1, 2), labels = c("D-penicillamine", "Placebo")),
    stage = factor(stage, levels = 1:4, labels = paste("Stage", 1:4))
  )

12.4.1 Wilcoxon Rank-Sum (Mann-Whitney): Two Independent Groups

pbc_rand <- pbc %>% filter(!is.na(trt), !is.na(bili))

# Visualise
pbc_rand %>%
  ggplot(aes(x = trt, y = bili, fill = trt)) +
  geom_boxplot(alpha = 0.7, outlier.shape = NA) +
  geom_jitter(width = 0.15, alpha = 0.3, size = 1.5) +
  scale_fill_manual(values = c("steelblue", "tomato")) +
  scale_y_log10() +
  labs(title = "Bilirubin by treatment group (log scale)",
       x = NULL, y = "Bilirubin (mg/dL, log scale)") +
  theme_bw() + theme(legend.position = "none")
wilcox.test(bili ~ trt, data = pbc_rand)

    Wilcoxon rank sum test with continuity correction

data:  bili by trt
W = 12007, p-value = 0.8422
alternative hypothesis: true location shift is not equal to 0
tidy(wilcox.test(bili ~ trt, data = pbc_rand))
# A tibble: 1 × 4
  statistic p.value method                                           alternative
      <dbl>   <dbl> <chr>                                            <chr>      
1     12007   0.842 Wilcoxon rank sum test with continuity correcti… two.sided

Interpreting the output: The p-value is from a permutation of ranks. The test statistic W is the number of times an observation from group 1 is greater than an observation from group 2. There is no mean difference in the output — report medians and IQR.

pbc_rand %>%
  group_by(trt) %>%
  summarise(median = median(bili), q1 = quantile(bili, 0.25), q3 = quantile(bili, 0.75))
# A tibble: 2 × 4
  trt             median    q1    q3
  <fct>            <dbl> <dbl> <dbl>
1 D-penicillamine    1.4 0.8     3.2
2 Placebo            1.3 0.725   3.6

12.4.2 Kruskal-Wallis: Three or More Groups

pbc_stage <- pbc %>% filter(!is.na(stage), !is.na(bili))

kruskal.test(bili ~ stage, data = pbc_stage)

    Kruskal-Wallis rank sum test

data:  bili by stage
Kruskal-Wallis chi-squared = 46.638, df = 3, p-value = 4.15e-10

The Kruskal-Wallis test tests whether at least one group has a different distribution. If the overall test is significant, follow up with pairwise Wilcoxon tests and Bonferroni correction:

pairwise.wilcox.test(pbc_stage$bili, pbc_stage$stage,
                     p.adjust.method = "bonferroni")

    Pairwise comparisons using Wilcoxon rank sum test with continuity correction 

data:  pbc_stage$bili and pbc_stage$stage 

        Stage 1 Stage 2 Stage 3
Stage 2 1.000   -       -      
Stage 3 0.029   0.168   -      
Stage 4 2.0e-05 2.1e-07 8.1e-05

P value adjustment method: bonferroni

12.5 Example 2: Animal Data (Small Experiment)

Suppose we measure tumour volume in 8 mice per group. With such small samples, normality cannot be verified, so a Wilcoxon test is appropriate.

set.seed(2024)
tumour <- tibble(
  group  = rep(c("Control", "Treatment"), each = 8),
  volume = c(
    rlnorm(8, meanlog = 3.5, sdlog = 0.8),   # right-skewed
    rlnorm(8, meanlog = 2.8, sdlog = 0.8)
  )
)

tumour %>%
  ggplot(aes(x = group, y = volume, fill = group)) +
  geom_boxplot(alpha = 0.7) +
  geom_jitter(width = 0.1, size = 3, alpha = 0.7) +
  scale_fill_manual(values = c("steelblue", "tomato")) +
  labs(title = "Tumour volume by treatment group (n=8/group)",
       x = NULL, y = "Tumour volume (mm�)") +
  theme_bw() + theme(legend.position = "none")
wilcox.test(volume ~ group, data = tumour)

    Wilcoxon rank sum exact test

data:  volume by group
W = 61, p-value = 0.001088
alternative hypothesis: true location shift is not equal to 0
# Compare with t-test (for illustration)
t.test(volume ~ group, data = tumour)

    Welch Two Sample t-test

data:  volume by group
t = 4.1394, df = 9.4678, p-value = 0.002263
alternative hypothesis: true difference in means between group Control and group Treatment is not equal to 0
95 percent confidence interval:
 18.69702 63.02147
sample estimates:
  mean in group Control mean in group Treatment 
               54.57411                13.71487

With only 8 observations per group and right-skewed data, the Wilcoxon test is more reliable. The t-test result should be treated with caution.

12.5.1 Paired Wilcoxon: Before-After Measurements

set.seed(2024)
before_after <- tibble(
  mouse   = 1:10,
  before  = rlnorm(10, meanlog = 3, sdlog = 0.5),
  after   = before * rnorm(10, mean = 0.7, sd = 0.2)  # treatment reduces volume
)

wilcox.test(before_after$after, before_after$before, paired = TRUE)

    Wilcoxon signed rank exact test

data:  before_after$after and before_after$before
V = 0, p-value = 0.001953
alternative hypothesis: true location shift is not equal to 0

12.7 What Can Go Wrong

Using nonparametric tests as a default “safe” choice Nonparametric tests are not always better. When data are normally distributed, the t-test is more powerful. Use nonparametric tests when (1) sample size is small and normality is suspect, (2) the outcome has extreme outliers that reflect real biology, or (3) the outcome is an ordinal scale.

Reporting the wrong summary statistics Wilcoxon tests compare distributions, not means. Report medians and IQR, not means and SD, alongside a Wilcoxon test result.

Treating a non-significant Kruskal-Wallis as proof of no difference A p > 0.05 from Kruskal-Wallis means the data are consistent with no group differences � but with small samples, power may be low. Report the effect size (e.g., median differences) and note if the study is underpowered.

Pairwise comparisons without correction If Kruskal-Wallis is significant, doing all pairwise Wilcoxon tests without correction inflates the Type I error. Always apply p.adjust.method = "bonferroni" or "holm" in pairwise.wilcox.test().

12.8.3 Exercise 3 (Open-ended)

From your own data, identify a continuous variable that is clearly non-normal (e.g., cytokine levels, survival times, cell counts). Run both a t-test and a Wilcoxon test. Do they agree? Which is more appropriate, and why?

12.9 Comprehension Check

  1. In what two situations is a nonparametric test preferred over a t-test?
  2. What does the Wilcoxon rank-sum test actually test? What is its null hypothesis?
  3. You run a Kruskal-Wallis test and get p = 0.02. What should you do next?
  4. Should you report means or medians alongside a Wilcoxon test result?
  5. Your colleague says “I used a Wilcoxon test because it doesn’t assume normality, so it’s always safer.” Is this correct?
Answers
  1. When (1) sample size is small (\(n < 30\)) and data are visibly non-normal, or (2) the data contain extreme outliers that represent real biology (not data errors).
  2. The Wilcoxon rank-sum test tests whether the two groups are drawn from the same distribution. More precisely, the null hypothesis is that the probability of an observation from group 1 being greater than a randomly selected observation from group 2 equals 0.5 (i.e., no tendency for one group to have larger values).
  3. Follow up with pairwise Wilcoxon tests (using pairwise.wilcox.test()) with a Bonferroni or Holm correction to identify which specific group pairs differ.
  4. Medians and IQR. The Wilcoxon test is based on ranks, and medians are the rank-based measure of location. Reporting means alongside a Wilcoxon test is inconsistent.
  5. Not quite. Nonparametric tests are more robust when normality is violated, but they are less powerful when data are approximately normal. Using them by default when a t-test would be valid loses statistical power unnecessarily.
