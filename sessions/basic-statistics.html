14.1 Introduction

This session introduces the essential statistical concepts used in clinical research. It focuses on interpretation, reproducible code, and how basic statistics support real research decisions.

14.2 Learning outcomes

At the end of this session you will be able to:

  • Summarize clinical data using descriptive statistics.
  • Visualize distributions and compare groups with clear plots.
  • Compute and interpret confidence intervals, t-tests, and ANOVA.
  • Fit basic regression and correlation models.
  • Use contingency tables for categorical clinical outcomes.
  • Apply power calculations for planning clinical studies.

14.3 Methods covered in this session

  • Descriptive statistics and summary measures
  • Data transformation and standardization
  • Confidence interval calculation and interpretation
  • One-sample and two-sample t-tests
  • Paired comparisons and Welch tests
  • Power and sample size reasoning

14.4 1. Why basic statistics matter in research

Clinical research begins with data that describe people, treatments, and outcomes. Before using complex models, researchers must answer simple questions:

  • What is the typical value of this lab measurement?
  • How variable is it across patients?
  • Do two groups have different average outcomes?
  • Are two clinical measurements correlated?
  • Can we quantify uncertainty around our estimates?

These questions are the foundation of credible research.

14.5 2. Descriptive statistics with clinical data

We begin with the pbc dataset, a standard clinical dataset from the survival package.

pbc <- survival::pbc %>%
  as_tibble() %>%
  janitor::clean_names() %>%
  mutate(
    sex = factor(sex, levels = c("f", "m"), labels = c("Female", "Male")),
    stage = factor(stage)
  )

pbc %>%
  summarize(
    n = n(),
    mean_bili = mean(bili, na.rm = TRUE),
    sd_bili = sd(bili, na.rm = TRUE),
    median_bili = median(bili, na.rm = TRUE),
    iqr_bili = IQR(bili, na.rm = TRUE)
  )
# A tibble: 1 × 5
      n mean_bili sd_bili median_bili iqr_bili
  <int>     <dbl>   <dbl>       <dbl>    <dbl>
1   418      3.22    4.41         1.4      2.6

Notice how we use na.rm = TRUE to ignore missing values. This is essential for research datasets where missingness is common.

14.6 3. Transformations and normality

Clinical variables are often skewed. Log-transforming can make a variable easier to interpret and analyze.

pbc <- pbc %>%
  mutate(log_bili = log(bili))

pbc %>%
  ggplot(aes(sample = log_bili)) +
  stat_qq() +
  stat_qq_line(color = "firebrick") +
  labs(
    title = "QQ plot of log-transformed bilirubin",
    x = "Theoretical quantiles",
    y = "Sample quantiles"
  ) +
  theme_minimal()

14.6.1 3.1 Interpretation

The Q-Q plot shows whether the log-transformed variable approximates a normal distribution. Research methods often rely on this assumption, so examining plots like these is good practice.

14.7 4. Confidence intervals

A confidence interval quantifies uncertainty around an estimate. In research, we use it to express how precise our estimate is.

ci_bili <- pbc %>%
  summarize(
    estimate = mean(log_bili, na.rm = TRUE),
    se = sd(log_bili, na.rm = TRUE) / sqrt(sum(!is.na(log_bili))),
    df = sum(!is.na(log_bili)) - 1
  ) %>%
  mutate(
    t_crit = qt(0.975, df),
    lower = estimate - t_crit * se,
    upper = estimate + t_crit * se,
    lower_orig = exp(lower),
    upper_orig = exp(upper),
    estimate_orig = exp(estimate)
  )

ci_bili
# A tibble: 1 × 9
  estimate     se    df t_crit lower upper lower_orig upper_orig estimate_orig
     <dbl>  <dbl> <dbl>  <dbl> <dbl> <dbl>      <dbl>      <dbl>         <dbl>
1    0.571 0.0501   417   1.97 0.473 0.670       1.60       1.95          1.77

This example shows a 95% confidence interval for the mean bilirubin on the original scale.

14.8 5. Comparing two groups with a t-test

The ToothGrowth dataset illustrates a basic comparison between two groups.

tg <- datasets::ToothGrowth %>%
  as_tibble() %>%
  mutate(dose = factor(dose))

tg %>%
  filter(dose == 0.5) %>%
  group_by(supp) %>%
  summarize(
    n = n(),
    mean_len = mean(len),
    sd_len = sd(len)
  )
# A tibble: 2 × 4
  supp      n mean_len sd_len
  <fct> <int>    <dbl>  <dbl>
1 OJ       10    13.2    4.46
2 VC       10     7.98   2.75
tg_low <- tg %>% filter(dose == 0.5)

test_result <- t.test(len ~ supp, data = tg_low)

broom::tidy(test_result)
# A tibble: 1 × 10
  estimate estimate1 estimate2 statistic p.value parameter conf.low conf.high
     <dbl>     <dbl>     <dbl>     <dbl>   <dbl>     <dbl>    <dbl>     <dbl>
1     5.25      13.2      7.98      3.17 0.00636      15.0     1.72      8.78
# ℹ 2 more variables: method <chr>, alternative <chr>

14.8.1 5.1 Interpretation

The t-test compares average tooth length between the two supplement groups. A smaller p-value suggests evidence of a true difference in means.

14.8.1.1 How to explain to a clinician

  • Report: the estimated mean difference and its 95% confidence interval (CI); say explicitly whether the CI includes zero.
  • Context: explain whether the observed difference is large enough to matter for patient care — focus on effect size and clinical relevance, not just the p-value.

14.9 6. One-way ANOVA

One-way ANOVA extends the comparison to three or more groups.

pg <- datasets::PlantGrowth %>% as_tibble()

pg_aov <- aov(weight ~ group, data = pg)
summary(pg_aov)
            Df Sum Sq Mean Sq F value Pr(>F)  
group        2  3.766  1.8832   4.846 0.0159 *
Residuals   27 10.492  0.3886                 
---
Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1

Use TukeyHSD() to see which groups differ.

TukeyHSD(pg_aov)
  Tukey multiple comparisons of means
    95% family-wise confidence level

Fit: aov(formula = weight ~ group, data = pg)

$group
            diff        lwr       upr     p adj
trt1-ctrl -0.371 -1.0622161 0.3202161 0.3908711
trt2-ctrl  0.494 -0.1972161 1.1852161 0.1979960
trt2-trt1  0.865  0.1737839 1.5562161 0.0120064

14.9.1 6.1 Interpretation

ANOVA tests whether at least one group mean differs from the others. Tukey’s test shows which specific group pairs are different.

14.10 7. Correlation and linear regression

Correlation measures the relationship between two continuous variables.

mtcars <- datasets::mtcars %>%
  as_tibble(rownames = "model") %>%
  mutate(am = factor(am, labels = c("Automatic", "Manual")))

correlation <- cor.test(mtcars$wt, mtcars$mpg)
correlation

    Pearson's product-moment correlation

data:  mtcars$wt and mtcars$mpg
t = -9.559, df = 30, p-value = 1.294e-10
alternative hypothesis: true correlation is not equal to 0
95 percent confidence interval:
 -0.9338264 -0.7440872
sample estimates:
       cor 
-0.8676594

A simple linear model quantifies the association.

lm_mpg <- lm(mpg ~ wt, data = mtcars)
broom::tidy(lm_mpg)
# A tibble: 2 × 5
  term        estimate std.error statistic  p.value
  <chr>          <dbl>     <dbl>     <dbl>    <dbl>
1 (Intercept)    37.3      1.88      19.9  8.24e-19
2 wt             -5.34     0.559     -9.56 1.29e-10

14.10.1 7.1 Interpretation

The regression slope tells us the expected change in mpg for each unit increase in weight.

14.10.1.1 How to explain to a clinician

  • Report: the slope as the expected change in outcome per unit change in the predictor, with its 95% CI (for example: “Each 1 kg increase in weight is associated with X change in mpg, 95% CI [L, U]”).
  • Context: state whether the magnitude is clinically meaningful and mention key covariates that were (or were not) adjusted for; avoid implying causation from a single unadjusted model.

14.11 8. Categorical analysis and chi-square tests

Contingency tables help analyze categorical outcomes.

hair_eye_table <- margin.table(HairEyeColor, c(1, 2))

hair_eye_table
       Eye
Hair    Brown Blue Hazel Green
  Black    68   20    15     5
  Brown   119   84    54    29
  Red      26   17    14    14
  Blond     7   94    10    16
chi_result <- stats::chisq.test(hair_eye_table)

chi_result

    Pearson's Chi-squared test

data:  hair_eye_table
X-squared = 138.29, df = 9, p-value < 2.2e-16

14.11.1 8.1 Interpretation

A chi-square test evaluates whether hair color and eye color are associated beyond random chance.

14.13 10. Exercises

  1. Create a table of mean bilirubin by sex in the pbc dataset.
  2. Fit a linear model predicting mpg from wt and interpret the slope.
  3. Use PlantGrowth to compute and interpret a Tukey pairwise comparison.
  4. Create a contingency table from HairEyeColor and explain the chi-square test result in one sentence.
  5. Compute a 95% confidence interval for mean bilirubin on the log scale, back-transform it to the original scale, and write one sentence describing what the interval says about typical patient bilirubin.

14.13.1 Exercise answers (hints)

  • Exercise 5 hint: compute mean and standard error on log_bili, use qt(0.975, df) to get the t critical value, build the CI on the log scale, then exp() the limits and the mean to return to the original scale; interpret the interval as the plausible range for the population mean bilirubin under the model assumptions.

14.14 Key takeaways

Basic statistics are not just formulas. They are tools for asking and answering clinical research questions with data. Every analysis should connect back to interpretation and research decisions.
